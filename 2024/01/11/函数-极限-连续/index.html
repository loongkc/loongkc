二、极限的性质

  1. 有界性

    （数列）如果数列 x n 收敛，那么是数列 x n 一定有界。（反之不成立）
    （函数）若 lim x→ x 0 f(x)存在，则f(x)在 x 0 某去心邻域有界（即局部有界）。（反之不成立）

  2. 保号性

    （数列） lim x→∞ x n =A （1）如果A>0(或A<0)，则N>0，当n>N时， x n >0(或 x n <0); （2）如果存在N>0，当n>N时， x n ≥0(或 x n ≤0)，则A≥0(或A≤0). 【注】注意结论（1）中是严格不等号(>和<);而（2）中是非严格不等号(≥和≤).
    （函数） lim x→ x 0 f(x)存在，则f(x)在 x 0 某去心邻域有界（即局部有界）。（反之不成立） （1）如果A>0(或A<0)，则存在δ>0，当x∈ U ∘ ( x 0 ,δ)时，f(x)>0(或f(x)<0); （2）如果存在δ>0，当x∈ U ∘ ( x 0 ,δ)时，f(x)≥0(或f(x)≤0)，则A≥0(或A≤0).

  3. 极限值与无穷小之间的关系

    limf(x)=A⇔f(x)=A+α(x), 其中limα(x)=0.
